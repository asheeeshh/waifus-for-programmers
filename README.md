## Run locally

Clone the repository
```bash
$ git clone https://github.com/asheeeshh/waifus-for-programmers.git
```
Install dependencies
```bash
$ cd waifus-for-programmers
$ npm i
```
Run the app
```bash
$ npm run dev
```
